This project is a RESTful API built with Flask and MongoDB that makes managing data secure and straightforward. It allows users to easily create, read, update, and delete information while ensuring that their accounts are protected through user authentication with JSON Web Tokens (JWT). With this setup, only verified users can access sensitive parts of the API.

Key Features:

User Registration and Login: Enjoy secure authentication with JWT, so users can safely manage their sessions.
CRUD Operations: Effortlessly create, read, update, and delete resources in the MongoDB database as needed.
Postman Integration: Use Postman for easy interaction with the API, making it simple to test and verify that everything works as it should.
API Testing with Pytest: Rely on thorough testing with Pytest to ensure all endpoints are reliable and perform well.
Error Handling: Benefit from clear error messages and status codes that guide users smoothly through any issues they might encounter.
